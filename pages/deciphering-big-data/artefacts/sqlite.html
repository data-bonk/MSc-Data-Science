SQLite

In preparation for the seminar, we were asked to complete an example involving storing data in a relational database, using the data from the Unit 4 Lecturecast activities.

The script I produced demonstrated a complete SQLite workflow for analysing the cleaned UNICEF survey data from the unit 4 processing pipeline. Here’s how it works:

  1. Database creation and data loading
    • The script establishes a connection to a SQLite database called ‘unicef.db’ (creating it if it doesn’t exist). It then reads the cleaned CSV file (‘mn_swapped_headers.csv’). This is the output from my header replacement script. It then loads it directly into the database using pandas’ to_sql() method.
  2. Key parameters in data loading:
    • ‘if_exists=’append’’: adds data to the table if it already exists, rather than overwriting.
    • ‘index=True’: Includes the pandas DataFrame index as a column in the database table.
  3. Query execution
    • The script creates a cursor object to execute SQL commands and performs a sample analytical query. It groups the survey respondents by religion and counts the frequency of each religious affiliation using 'GROUP ’Y' and ‘COUNT()’ functions. This demonstrated how the descriptive column headers from my cleaning process make the data immediately queryable with meaningful field names.
  4. Resource Management
    • My script follows good database practices by properly closing both the cursor and connection objects after use. Additionally, it includes a cleanup step that removes the temporary database file using the ‘os.remove()’ function, which helps test scenarios or when persistent storage is not needed.
  5. Workflow integration
    • My script effectively completes the data processing pipeline by transforming the human-readable survey data into a format suitable for SQL-based analysis. The combination of pandas for data manipulation and SQLite for querying provides a powerful toolkit for exploring survey patterns and relationships within the UNICEF dataset.
Code
import sqlite3
import pandas as pd
import os

# connect/create CB
con = sqlite3.connect('unicef.db')

# Read CSV
df = pd.read_csv('mn_swapped_headers.csv')
# Convert to SQL query
sql = df.to_sql('unicef', con, if_exists='append', index=True)

# Define cursor
cur = con.cursor()

# Simple Query + Output
res = cur.execute('SELECT religion, COUNT(0) FROM unicef GROUP BY religion')
print([r for r in res])

# Close connection
cur.close()
con.close()

# Remove Database
os.remove('unicef.db')
