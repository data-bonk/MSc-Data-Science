Web Scraping

For unit 3, we were tasked with a web scraping activity focused on the word “Data Scientist”. For this, I chose to perform a targeted web scraping of a data science report from Stitch Data’s website.

Here is how it works:
Data extraction:

My script makes an HTTP GET request to retrieve the webpage content, then used BeautifulSoup to parse the HTML. It targets two types of content containers:

  1. Key Findings Extraction. The code searches for ‘div; elements with the class ‘key-findings-box-. For each finding box, it extracts two paragraph elements – the first containing the title and the second containing descriptive information. These are stored as dictionaries with ‘title’ and ‘info’ keys.
  2. Statistics extraction. Similarly, it locates ‘div’ elements with the class ‘stats-box’ and extracts statistical data. The script pulls the numeric value from ‘h3’ tags and the description from ‘p’ tags. It includes logic to handle formatting, converting ‘k’ suffices to thousands and processing decimal values appropriately.
Data processing features:
  • Text cleaning: it uses ‘.get_text()’ to extract clean text content from HTML elements.
  • Numeric conversion: transforms string values like “15k” into proper numeric format (15000).
  • Structured storage: organised extracted data into a structured format with findings as a list of dictionaries and statistics as a single dictionary.
Output generation:

The processed data is compiled into a JSON structure containing both findings and statistics, then written to a file called “scraping_output.json”

Key Strengths:
My implementation demonstrated good practices by using specific CSS selectors to target exact content areas rather than scraping entire pages. The numeric processing logic handles common web formatting conventions, and the structured output format makes the data easily accessible for further analysis. This approach is particularly effective for extracting structured insights from data science industry reports, creating a reusable dataset for analysis or research purposes.
Code:

import requests
import json
from bs4 import BeautifulSoup
target_url = 'https://www.stitchdata.com/resources/the-state-of-data-science/'

# Make GET request to page, grab content, store in page_contents
page_contents = requests.get(target_url).content

# Initialise beautiful soup with the contents
soup = BeautifulSoup(page_contents, features="html.parser")

# Selector on page contents to find content
findings_boxes = soup.find_all('div', 'key-findings-box')

compiled_findings = []

# For each section of content, extract the title and info and store in the compiled_findings list as a dict
for finding_box in findings_boxes:
    [title, info] = finding_box.find_all('p')
    compiled_findings.append({'title': title.get_text(), 'info': info.get_text()})

# Selector on page to find statistics
stats_boxes = soup.find_all('div', 'stats-box')

# Initialise dict
compiled_stats = {}

# for each stat box, extract title and value, convert to a float and store.
for stats_box in stats_boxes:
    [value] = stats_box.find_all('h3')
    [title] = stats_box.find_all('p')
    numeric_value = value.get_text().replace('k', '000')
    if ('.' in numeric_value):
        numeric_value = float(numeric_value) * 1000
    else:
        numeric_value = float(numeric_value)
    compiled_stats[title.get_text()] = numeric_value

# Output to file
with open("scraping_output.json", "w") as f:
  f.write(json.dumps({ 'findings': compiled_findings, 'stats': compiled_stats }))
